Understanding Insertion sort in C

I’ve recently been working on a coding game that required implementing the insertion sort algorithm. It was fascinating to see how this simple algorithm can efficiently sort data. In this article, I’ll walk you through the code step-by-step to make it easily comprehensible and visual.

The task is to find the correct position in an array to insert a number in a sorted, decreasing order. Here’s an example of the possible input and the required output:

Input:
5
34
23
45
12
30
output:
sorted_array: [45, 34, 30, 23, 12]

Initial Setup:

First, let’s create required variables and arrays:

    int N = 5;
    int intput_array[] = {5, 34, 23, 45, 12, 30}
    
    int sorted_array[N];

Insertion Sort Algorithm:

Now let’s go over each element one by one, positioning them in the correct order in the freshly created array named sorted_array:

    for (int i = 0; i < N; i++) {
        int X = input_array[i]; /* one element at a time */
        
        int j = i; /* j is the variable for findinig the right position*/
        while (j > 0 && sorted_array[j-1] < X) {
            sorted_array[j] = sorted_array[j-1]; /* shifting elements */
            j--;
        }
        sorted_array[j] = X;
    }

I’m a big fan of this part I found it quite impressive of much “dancing around” we can observe. Despite involving multiple movements, it effectively places each element in its correct position within the sorted array.

Analysis:

In “The C Programming Language” by Kernighan and Ritchie, there is a quote:

“A sort often consists of three parts - a comparison that determines the ordering of any pair of objects, an exchange that reverses their order, and a sorting algorithm that makes comparisons and exchanges until the objects are in order.”

Let’s break down these three parts in the context of our example:

  • Comparison: The while loop condition (j > 0 && sorted_array[j-1] < X) determines the ordering of elements.
  • Exchange: Inside the while loop, the assignment (sorted_array[j] = sorted_array[j-1]) shifts the smaller element one step to the right.
  • Sorting Algorithm: The use of the variable j to find the correct position for X, decrementing it as needed until X is placed correctly.
